# Dotfiles
Sets up Ubuntu development environment using ansible.

# Installation
```
git clone git@github.com:binarymason/dotfiles.git ~/.files
cd ~/.files
./script/setup
```

# Running Tests

With Docker

```
./script/cibuild
```

On bare metal
```
./script/test
```
